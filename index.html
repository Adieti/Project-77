Open Console


Visual Studio Code is an IDE from Microsoft. It is used to develop computer programs, websites, web apps, web services and mobile apps


A console refers to a computer terminal where a user may input commands and view output such as the results of the commands or status messages from the computer.
You can open the console in Chrome by right-clicking on any webpage, then click on inspect and the console screen will appear.
To print any text in the console , you will need to use the "console.log()" command.
